Comments.
  Data in this portion is re-reduction of observations available in the following portions:
  sm0010, sm0012, sm0013, sm0014, sm0016, sm0017, sm0025, sm0027, sm0058, sm0064, sm0075.
  Note, however, that it is better for the user to check if these portions contain unique 
  observations which, for some reason, were not reduced to Gaia DR2. In particular, this
  portion does not include observations made in 2008 available in sm0058.
